# Traffic Line Optimizer

A Python-based optimization tool that balances SMS client traffic across multiple routing lines using historical traffic data and a greedy load balancing algorithm.

---

## Features

- Reads SMS traffic data from CSV
- Reads client list from Excel
- Cleans and validates datasets
- Creates daily traffic pivot tables
- Generates client traffic statistics
- Assigns clients to routing lines using a Greedy Load Balancing algorithm
- Exports optimized routing to Excel

---
